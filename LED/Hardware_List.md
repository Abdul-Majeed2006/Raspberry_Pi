## Essential (Required)

### 1. Raspberry Pi Pico
- **Quantity**: 1
- **Specs**: Any Raspberry Pi Pico or Pico H (with pre-soldered headers)
- **Where**: Amazon, Adafruit, Pimoroni
- **Price**: ~$4-6 USD

### 2. Micro USB Cable
- **Quantity**: 1
- **Purpose**: Power + Programming
- **Note**: Most phone chargers work fine

### 3. Breadboard
- **Quantity**: 1
- **Size**: 400 or 830 tie-points
- **Purpose**: No soldering needed, plug-and-play wiring

### 4. Jumper Wires (Male-to-Male)
- **Quantity**: ~20 pack
- **Purpose**: Connecting components on breadboard

## Components (For Exercises)

### 5. LEDs
- **Quantity**: 
  - 5x Standard LEDs (any color)
  - 1x RGB LED (Common Cathode)
- **Note**: Most kits include resistors too

### 8. Potentiometer

### 9. Photoresistor (LDR)

- **Purpose**: Light sensor for Night Light project
